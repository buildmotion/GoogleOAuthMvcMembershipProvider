Thanks for downloading and using the Vergoisty Framework. To begin using the framework, enable the license by doing the following:

	1. locate the buildmotion.lic file. See the [content] folder. You may need to add the item to the project.

	2. Show or display the properties for the license file.

	3. Set the option [Copy to Output Directory] to [Copy Always].

Note: The [Build Action] should be set to [None].
